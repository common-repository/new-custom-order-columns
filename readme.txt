== Installation ==

1. Upload "new-custom-order-columns.zip" to the "/wp-content/plugins/" directory.
2. Activate the plugin through the "Plugins" menu in WordPress.
3. OR, Go to WP admin panel, click 'Plugins' -> 'Add new'. In the search input box, type 'new custom order columns'.
4. Install and activate the plugin.
5. Plugin settings found beside activate/deactivate plugin on plugins page, also found on admin menu under woocommerce menu.
